# Medical Students Data Analytics using Python

## Project Overview

This project focuses on performing end-to-end data analysis on a Medical Students dataset using Python. The objective is to clean the dataset, handle missing values, remove duplicates and outliers, perform exploratory data analysis (EDA), and generate meaningful insights through visualizations.

---

## Objectives

- Understand the dataset structure
- Handle missing values
- Remove duplicate records
- Detect and treat outliers
- Create new features (BMI Category)
- Analyze health-related patterns
- Visualize important insights


## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab Notebook


## Project Workflow

### Data Cleaning
- Checked dataset shape
- Identified missing values
- Filled missing values using Mean, Forward Fill (FFill), and Backward Fill (BFill)
- Removed duplicate records
- Detected and handled outliers using the IQR method


### Feature Engineering
Created a new BMI Category column:

- Underweight (BMI < 18.5)
- Normal (18.5–24.9)
- Overweight (25–30)
- Obesity (>30)


## Exploratory Data Analysis (EDA)

The project answers the following business questions:

- Dataset dimensions (rows and columns)
- Missing value analysis
- Numerical and categorical column identification
- Duplicate detection
- Outlier analysis
- Average blood pressure by gender
- Diabetes distribution among males and females
- Cholesterol comparison by gender
- Smoking habits comparison
- Smoking vs Diabetes analysis
- Blood group percentage distribution
- Diabetes across age groups
- Average heart rate of diabetic and non-diabetic students
- Relationship between Heart Rate, Blood Pressure, and Diabetes
